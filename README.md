# 🎯 Instagram Short Username Finder

A Python tool that finds **available 1-4 character Instagram usernames** automatically. No manual checking required.

## ✨ Features

- Generates all valid 1-4 character usernames
- Checks availability on Instagram in real-time
- Supports letters, numbers, underscores (`_`), and periods (`.`)
- Multi-threaded for fast checking
- Saves results to a file automatically
- Shows a clean list of available usernames at the end

## 📋 Requirements

- Python 3.6+
- Internet connection
